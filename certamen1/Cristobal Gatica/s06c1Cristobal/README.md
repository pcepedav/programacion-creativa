##### Descripción (SE RECOMIENDA INICIAR MAS DE UNA VEZ EL PROGRAMA PARA VER MAS RESULTADOS)

Este programa es el resultado de una referencia de trabajo de "RadarBoy3000" un diseñador experimental de escultura en data. Tomando las bases de su proyeccto "Growing a node" mi resultado no fue el esperado pero tampoco una lejania
de este, el programa funciona a base de ellipses con direcciones aleatoreas las cuales al estar entre 850 puntos de distancia estas forman una linea entre ellas la cual va variando en una escala de grises formando un efecto
de opacidad. Los puntos actuan formando patroneas aleatoreos.

Al presionar el mouse se sobrepone una capa nueva de random(grises) para dar distintos tipos de fondos, esto es solo una capa nueva por lo que no reinicia el control aleatoreo de patrones.
