Certamen 01

El propósito de este trabajo consiste en la exploración del movimiento dentro de la programación creativa. Este código fue escrito pensando en la representación visual de un agujero negro, con un centro que aparenta absorber sus alrededores. Este código cuenta con la posibilidad de interactuar con la pieza visual y crear una versión propia del usuario de un agujero negro al entregar herramientas para ajustar valores de las variables involucradas. A pesar de ser una pieza interactiva, este código puede generar una visualización propia si no existe ninguna acción del usuario.

Comandos:
 
Botón Izquierdo del Ratón: Aumento de Distancia del Centro
Botón Derecho del Ratón: Reducción de Distancia del Centro
Tecla "W": Aumento de Velocidad de Rotación
Tecla "S": Reducción de Velocidad de Rotación 
Tecla "G": Aumento de Tamaño de Círculos 
Tecla "P": Reducción de Tamaño de Círculos
Tecla "1": Comando de Grosor 1
Tecla "2": Comando de Grosor 2
Tecla "3": Comando de Grosor 3
