Para la exploración y desarrollo se analizó el siguiente referente solo en el ambito visual, el codigo
esta generado a base de mis conocimientos sin uso del codigo del referente:

http://piratefsh.github.io/presentations/beauty-of-algorithms/slides/index.html#/


Se eligió el referente por como los objetos presentes en el progama interactuan entre si junto con un interés personal
por la composicion que se desarrolla, de caracter minimalista y de armonía entre objetos.

En el referente se observa como varias particulas interactuan entre sí y cuando 3 de estas se acercan
lo suficiente entre ellas se genera un relleno del espacio entre particulas.

Para lograr este resultado se creó una clase para dichas particulas, en este caso "Punto" del cual se generaron 3 objetos
y los metodos que posee dicha clase le permiten a los objetos moverse en el espacio, interactuar con los bordes y entre sí,

A medida que se acorta la distancia entre cada objeto se genera una linea entre estos y cuando los 3 objetos se encuentran
a una corta distancia entre sí se puede apreciar el relleno de la figura generada entre los 3 objetos.

La opacidad de dicho relleno varía segun el tiempo que se mantienen los 3 objetos a la distancia definida.
